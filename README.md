# 🔥 Roast My GitHub

Type a GitHub username, pick a roast style, and get a witty, data-accurate roast of someone's public repos — rendered as a hand-drawn developer's notebook.

Built with **Next.js (App Router) · TypeScript · Tailwind CSS**, roasts generated by **Claude**, data from the **GitHub REST API**.

> Built as a test task. The full set of prompts I used to drive Claude Code is pasted near the bottom — see [**Prompts used**](#-prompts-used-show-your-work).

---

## What it does

- **Type any GitHub username** → fetches their public profile + repos via the GitHub API.
- **Pick a roast style** — five distinct voices:
  | Style | Voice |
  |---|---|
  | **Corporate** | Passive-aggressive performance-review jargon |
  | **Shqiptarski** | Albanian dark humor — *besa*, Kanun, mountains, bunkers, diaspora; written entirely in Albanian |
  | **Haiku** | Two 5-7-5 haikus of quiet devastation |
  | **Gen Z** | "no cap, lowkey brutal" chaotic lowercase |
  | **Shakespearean** | Early Modern English, with a Bard reference in the burn |
- **The whole UI speaks in the selected voice** — labels, loading text, placeholders, *and every error message* change tone with the mode. (Haiku errors are themselves valid haikus. Shqiptarski errors are in Albanian.)
- **Graceful everything** — missing users, private/non-existent accounts, rate limits, network timeouts, and malformed input all produce friendly, in-character messages instead of crashes.
- **Runs with zero API keys.** No `ANTHROPIC_API_KEY`? The app drops into **Demo Mode** and serves hand-written template roasts personalized with real profile data. No `GITHUB_TOKEN`? It uses unauthenticated GitHub access (60 req/hr).

### The thoughtful touches
- **Sketchbook aesthetic** — paper texture, ruled lines, SVG-distorted "hand-drawn" borders, slight rotations, torn-paper result card.
- **Typewriter effect** — the roast "writes itself" in ink, with a blinking cursor.
- **Mode-aware loading & empty states** — e.g. Gen Z shows *"cooking ur roast rn..."*, Shakespeare shows *"the bard doth compose thy roast"*.
- **Demo Mode is a feature, not a failure** — the app is fully usable on a fresh clone with no setup at all.

## 🚀 Run it from a fresh clone (under 5 minutes)

**Prerequisites:** Node.js 18.18+ (tested on Node 20 & 23).

```bash
# 1. Clone
git clone https://github.com/rinormaliqi/roast-my-github.git
cd roast-my-github

# 2. Install
npm install

# 3. (Optional) add API keys — the app runs fine WITHOUT this step
cp .env.local.example .env.local
#    then open .env.local and paste your keys if you have them

# 4. Run
npm run dev
```

Open **http://localhost:3000** and roast away.

> **No keys? No problem.** Skip step 3 entirely and the app runs in **Demo Mode** with template roasts. Add an `ANTHROPIC_API_KEY` later for live Claude-generated roasts, and/or a `GITHUB_TOKEN` to lift the GitHub rate limit.

### Environment variables

| Variable | Required? | What it does |
|---|---|---|
| `ANTHROPIC_API_KEY` | Optional | Enables real Claude roasts. Without it → Demo Mode (template roasts). Get one at [console.anthropic.com](https://console.anthropic.com). |
| `GITHUB_TOKEN` | Optional | Raises GitHub rate limit from 60 → 5,000 req/hr. Without it → unauthenticated access still works. Create at [github.com/settings/tokens](https://github.com/settings/tokens) (no scopes needed). |

## ▶️ Run on Replit

1. **Create Repl → Import from GitHub** → paste this repo's URL.
2. Add your keys in the **Secrets** (🔒) panel: `ANTHROPIC_API_KEY` and optionally `GITHUB_TOKEN`. *(Skip this and it still runs in Demo Mode.)*
3. Set the run command to `npm run dev` (or `npm run build && npm run start` for production).
4. Press **Run** — Replit gives you a public URL anyone can open.

## How it's built

```
app/
  api/roast/route.ts   POST endpoint: validate → fetch GitHub → generate roast
  layout.tsx           Fonts + the SVG "rough" filter used for hand-drawn borders
  page.tsx             Server component: notebook shell (header / footer)
  globals.css          The entire sketchbook design system
components/
  NotebookContent.tsx  Client component: owns state, renders the two-page form + result
  RoastCard.tsx        The torn-paper result card with the typewriter effect
hooks/
  useTypewriter.ts     Adaptive-speed "writes itself" effect
lib/
  github.ts            GitHub API client — typed errors, timeouts, rate-limit handling
  roast.ts             Claude integration + automatic fallback to mock roasts
  mock-roast.ts        Offline template roasts (Demo Mode) for all 5 styles
  tone.ts              Single source of truth for all mode-specific UI copy
types/
  index.ts             Shared types (RoastStyle, RoastResponse, ApiError, ...)
```

**Design decisions worth noting:**
- **API routes, not a separate backend.** The Anthropic key never touches the client; all GitHub + Claude calls happen server-side in `app/api/roast/route.ts`.
- **The Anthropic client is lazily imported** so the app never crashes at startup when no key is set.
- **Layered fallback:** no key → mock roast; key present but Claude call fails → caught and falls back to mock roast (with a console warning) rather than erroring the user.
- **`lib/tone.ts` centralizes every UI string per style**, so the form's voice changes the instant you switch modes, and the result card keeps the voice of the roast it actually displayed.

## What I'd do with more time

- **Caching** — memoize roasts per `(username, style)` so repeat requests are instant and don't re-burn API quota.
- **Streaming** — stream Claude's tokens to the client so the typewriter reflects real generation instead of a post-hoc effect.
- **Shareable roast cards** — generate an OG image / "download as sticker" so a roast can be shared on social.
- **More signal in the roast** — commit frequency, README quality, issue/PR ratios, "longest-abandoned repo" for sharper, more specific burns.
- **Rate-limit UX** — show remaining GitHub quota and a live countdown to reset.
- **Tests** — unit tests for `github.ts` error mapping and `mock-roast.ts`, plus a Playwright happy-path.
- **i18n for Shqiptarski** — a full Albanian UI toggle, not just the roast voice.
